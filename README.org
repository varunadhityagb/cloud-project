#+title: Dynamic Carbon Profiling System

* How to Run

** Run the Minikube Script
#+begin_src bash
./rebuild-minikube
#+end_src

** Get all info about Pods and make sure all of them are ~Running~
#+begin_src bash
kubectl get pods -n carbon-profiling
#+end_src

** Run the Device Agent
#+begin_src bash
cd device-agent/
python device_agent.py             
#+end_src

** Open the [[http://192.168.49.2:30800/][Carbon-Aware Computing Dashboard]]
